# -성남시 공공데이터 시각화 경진대회 최종
주제 : 기초시설(학교, 병원, 경찰서)을 증가하여 걷는 생활권 확립, 이를 통하여 교통 체증 완화

### **1. 프로젝트 개요**

성남시 구시가지와 신시가지 간의 복지 인프라 격차로 인해 발생하는 불필요한 장거리 이동과 교통 혼잡 문제를 해결하기 위한 데이터 기반 공간 최적화 프로젝트입니다.

• **문제 정의:** 행정동 간 필수 시설(학교, 병원, 복지관 등) 쏠림 현상으로 인한 타 지역 원정 서비스 이용 및 간선도로 교통 체증 유발.

• **분석 목표:** 객관적인 데이터를 활용하여 복지 사각지대를 도출하고, 거주지 근거리에서 필수 인프라를 이용할 수 있는 최적의 신규 입지 좌표를 산출.

• **기대 가치:** 신규 시설 확충 시 기존 장거리 통행 수요를 국지적으로 흡수하여, 전체 통행 거리 및 차량 주행 거리(VKT)를 감소시키고 탄소 배출 저감 효과를 정량적으로 증명.

### **2. 분석 환경 및 활용 데이터**
**기술 스택:** Python (Pandas, Folium, Scikit-learn, PySAL), R, MySQL
**주요 데이터셋:**
| 데이터 카테고리    | 주요 변수 (Features) | 출처 | 비고 |
| ---------- | -------------------------------- |-----------|-----------|
| 복지 인프라 | 유치원/초/중/고, 병의원, 파출소 위치(위경도)        |공공데이터포털|위치 좌표 추출 필수|
| 인구 및 지리 | 동별 연령별 인구수, 행정동 면적, GeoJSON 경계 |통계청(KOSIS), 성남시	| |
| 교통 데이터 | 행정동 단위 기종점(OD) 통행량, 주요 간선도로 통행 속도 |성남공공데이터,ITS	| |
